# barren_creek

Fullstack project with a React (Vite) frontend and a Node/Express backend.

## Structure

```
barren_creek/
├── client/   # React frontend (Vite)
└── server/   # Node/Express backend
```

## Getting Started

### Backend

```bash
cd server
npm run dev   # starts on http://localhost:3001
```

### Frontend

```bash
cd client
npm run dev   # starts on http://localhost:5173
```
